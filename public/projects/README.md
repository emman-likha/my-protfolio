## How to Create Screenshots:

1. **Browser Extension Method:**
   - Use a browser extension like "Full Page Screen Capture" or "Awesome Screenshot"
   - Visit each website and capture a full-page screenshot
   - Save as JPG or PNG format

2. **Online Tools:**
   - Use services like:
     - https://screenshot.rocks/
     - https://www.screenshotapi.net/
     - https://htmlcsstoimage.com/

3. **Manual Method:**
   - Open each website in your browser
   - Take a screenshot (Windows: Win + Shift + S, Mac: Cmd + Shift + 4)
   - Crop to show the main content area
   - Save as JPG format

## Image Specifications:

- **Format:** JPG or PNG
- **Aspect Ratio:** 16:9 (recommended for consistency)
- **Resolution:** 1920x1080 or higher
- **File Size:** Keep under 500KB for optimal loading

## Note:

If images are not added, the portfolio will automatically use gradient placeholders as fallbacks.
